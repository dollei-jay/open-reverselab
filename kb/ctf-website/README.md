# Web CTF Knowledge Base

Web 安全 / CTF 技术知识库。

## 目录

| 目录 | 内容 |
|---|---|
| `techniques/` | 按攻击类型组织的技术文件 |
| `checklists/` | 操作 checklist |
| `payloads/` | 可复用 payload 种子 |

## 入口

- 攻击网络总览：`techniques/attack-network.md`
- 技术索引：`techniques/kb-index.json`
- 30 分钟快速 checklist：`checklists/web-ctf-first-30-min.md`
